Programming Languages - Terms and Definitions

Term Definition
Machine Language The lowest-level programming language, consisting of binary code that the computer's hardware can directly execute.
Debugging The process of identifying and fixing errors or bugs in software.
Assembler A program that converts assembly language code into machine code.
Assembly Language A low-level programming language that uses symbolic code and is closely related to machine language.
Translator A program that converts high-level programming code into machine-readable code.
Compiler A program that translates an entire high-level programming language into machine code before execution.
Interpreter A program that translates high-level programming code into machine code line by line as the program runs.
Cross-Platform Software Software that can run on multiple operating systems without modification.
Natural Language A language spoken or written by humans, such as English or Spanish.
Formal Language A language used in mathematics, logic, and computing that has a defined structure and rules.
Machine Independence The ability of software to run on different types of hardware without modification.
High-Level Programming Language A programming language that is easy for humans to read and write, abstracting away the details of the hardware.
Object-Oriented Programming (OOP) A programming paradigm based on the concept of objects that contain data and methods.
Scripting Language A high-level programming language used for automating tasks and writing small programs, often interpreted rather than compiled.
Data Types The classification of data based on its nature and the operations that can be performed on it. Examples include integers, floats, and strings.
Integer A whole number, either positive, negative, or zero, without a fractional part.
Float A number that can represent a fraction or decimal point.
String A sequence of characters, such as words or sentences, used to represent text.
Primitive Data Type A basic data type that is provided by a programming language, such as integers, floats, or booleans.
Data Structure A way of organizing and storing data to perform operations efficiently.
Python Collection Data Structures Built-in types in Python to store collections of data, including lists, tuples, sets, and dictionaries.
List A collection of ordered items in Python that can be changed after creation.
Tuple A collection of ordered items in Python that cannot be changed after creation.
Set A collection of unique items in Python.
Dictionary A collection of key-value pairs in Python.
Function A block of code in Python that performs a specific task when called.
Local Variable A variable defined within a function or block and accessible only within that scope.
Global Variable A variable defined outside any function, accessible throughout the program.
Source Program The original code written by a programmer in a high-level programming language before it is compiled or interpreted.
